# CODET-version-1.1
Time series at 28.4 nm and 21.1 nm predictions until March 19 2026 using model parameters from the CODET model version 1.1.

Modeling Solar Spectral Irradiance (SSI) from Iron Lines Using the CODET Model Version 1.1. 
The Astrophysical Journal, 2025  https://doi.org/10.3847/1538-4357/adcec6


Time interval from July 01 1996 to March 19 2026.

File description:

    SSI_CODETv11_284.csv => SSI at 28.4nm from CODET model version 1.1
    SSI_CODETv11_211.csv => SSI at 21.1nm from CODET model version 1.1
    SSI_codetv11_predictions_until_march_2026.png => Figure. Solar spectral irradiance (SSI) 
    prediction using COronal DEnsity and Temperature (CODET) model v1.1 from 1996 July 01 to 
    2024 October 28. Top: SSI from CODET v1.1 at 28.4 nm (black line), EVE/SDO (red line), 
    and EUVS/GOES (blue line). Bottom: SSI from CODET v1.1 at 21.1 nm (black line) and 
    EVE/SDO (magenta line); AIA daily mean values (green line) were overplotted.
